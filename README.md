# She_Can_Foundation
🌸 She Can Foundation — Frontend Development Internship Project

A modern and responsive NGO website created as part of a Frontend Development Internship project for She Can Foundation.
This project focuses on building an engaging digital experience that reflects the foundation’s mission of empowering women through education, mentorship, and community support.

✨ Project Overview

The website is designed to provide:

A visually appealing user experience
Smooth animations and modern interactions
Responsive layouts for all devices
Interactive sections that improve engagement

Built completely using HTML, CSS, and Vanilla JavaScript without any frontend frameworks.

🚀 Features
🎨 Modern UI/UX design
🌙 Dark / Light theme toggle
💫 Smooth scroll animations
🖱️ Custom animated cursor
📱 Fully responsive design
⌨️ Animated typewriter effect
💖 Interactive donation modal
📊 Animated counters
🎠 Quote carousel
📰 Newsletter subscription section
🎴 3D card tilt interactions
🌸 Beautiful gradient effects and typography
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
